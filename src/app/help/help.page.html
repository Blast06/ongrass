Ayuda

¿Cómo se conecta la app con el sistema?

La app se conecta a través de una conexión a internet, para interactuar directamente con el sistema de riego si tienes problemas al conectar contáctanos a través de nuestro correo on_grass@outlook.com

¿Como saber si no esta fallando?

Ongrass te mostrará alertas si hay algun fallo. Por ejemplo: si la bomba no se encendio te avisara que no hay conexion al sistema, asi podras verificar que todo este correcto, si las fallas continuan, reinicia la aplicacion.

¿Como saber si la humedad es correcta?

El sistema esta capacitado para automaticamente detectar valores claves para su funcionamiento, igual en la parte de Monitoreo veras informacion relacionada a que tanta humedad debe Existir para funcionar

¿Existe riesgos de usar Ongrass?

Definitivamente no Ongrass esta diseñado para si no se cumplen los parametros no funcionara la bomba que riega el agua al huerto, no debe preocuparse por su funcionamiento
